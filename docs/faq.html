GtL FAQ

Here are our frequently asked questions.

  1. What systems is this compatible with?

    2. Gateway is compatible with Windows, Mac, and Linux/GNU machines.

  2. How do I reset my control scheme?

    3. On the main menu, press Ctrl+R and the control scheme will reset to default (WASD + Space + Left Ctrl).

  3. How do I install a map-pack?

    4. First, properly download and install the map pack; copy all the folders from the downloaded map-pack into the Gateway's directory. Then, launch it and select the Play option, then navigate to the map pack. Map-pack titles will be displayed on screen, but you might have to press left & right to find the right page your map pack is on. Select the map pack and have fun!

  4. How do you create a map-pack? What sort of standards should I adhere to?

    5. I made a whole page about this! See this page for a comprehensive guide. For more information about map-packs, see the wiki page.
