Gateway to Legend

Here are our frequently asked questions.

  1. What systems is this compatible with?

    2. Gateway is compatible with Windows, Mac, and Linux machines.

  2. How do you create a map-pack? What sort of standards should I adhere to?

    3. Select "New" from the menu after launching the toolchain. Input all the data it asks. After that, a window will launch with your tileset on it, and instructions. Navigate the cursor to the tile it asks you to find and press Space on it. Finish and it will naturally quit. If you make a mistake, press Enter before you complete the map-pack, and try again with Map-Pack Wizard -> Edit Tile Equates.
    Maps are loaded in like such: first map -> top-most left-most map, second -> top-most next to left-most map, etc., wrapping around every 10. Map #10 is the right-most map of the first row, and #11 is left-most map of the next row.
    The tileset should be made of 48x48 tiles, and it should be formatted with rows of 8 tiles and columns of 16, so the size of the tileset image is 768 x 384 px. For more information, see the wiki page.

  3. How do I reset my control scheme?

    4. On the main menu, press Ctrl+R and the control scheme will reset to default (WASD + Space + Left Ctrl).
